What is HTML?

HTML stands for HyperText Markup Language. It is a markup language used to create web pages. HTML uses tags to mark up the text in a web page, which tells the browser how to display the text.

HTML tags are enclosed in angle brackets (<>). The tag name is always in lowercase letters. The tag name is followed by an optional attribute name and value. The attribute name is always in lowercase letters, and the attribute value is enclosed in quotation marks.

For example, the following code would create a heading tag:

<h1>This is a heading</h1>

The <h1> tag is the tag name, and the This is a heading is the text that will be displayed as the heading.

What is Semantic HTML?

Semantic HTML is a way of using HTML tags to describe the meaning of the content in a web page. This makes it easier for search engines to understand the content of the page and to rank it higher in search results.

Semantic HTML uses tags that have a specific meaning. For example, the <article> tag is used to mark up an article, and the <nav> tag is used to mark up a navigation bar.

Using semantic HTML can also make your web pages more accessible to people with disabilities. Screen readers can use the semantic tags to read the content of the page to people with disabilities.

Commonly Used Semantic Tags

  • <article> - Used to mark up an article
  • <aside> - Used to mark up an aside section
  • <header> - Used to mark up the header of a page
  • <nav> - Used to mark up a navigation bar
  • <section> - Used to mark up a section of a page
  • <footer> - Used to mark up the footer of a page
