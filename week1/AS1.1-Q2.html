Comments

HTML comments are the way to add notes to the html code that are not displayed on the browser. The HTML comments can be displayes by < !- - and - -> tags
< !- - This is a comment - ->

This is a paragraph.

The comments can be used for varity of purposes such as
->Explaining the purpose of a particular piece of code
->Documenting the code for future reference
->Leaving notes to yourself or other developers
->Hiding code that you do not want to be displayed in the browser
The comments are a valuable tool for making your HTML code more readable and maintainable. They can help you to understand your code better and to make changes more easily.
HTML comments can be nested, but only to a depth of 7 levels.
HTML comments can be multi-line, but they must be terminated with a --> tag on each line.
HTML comments can be used to hide code from the browser, but they cannot be used to prevent the browser from executing the code.
HTML comments are not processed by the browser, so they do not affect the appearance of the web page.
