HTML Tags

  1. html - The root element of an HTML document.
  2. head - Contains metadata about the document, such as the title, author, and description.
  3. title - Contains the title of the document.
  4. body - Contains the content of the document, such as text, images, and links.
  5. h1 to h6 - Used to create headings of different sizes.
  6. p - Used to create paragraphs.
  7. br - Used to create a line break.
  8. hr - Used to create a horizontal line.
