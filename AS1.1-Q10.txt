What are global attributes? List any 5 global attributes.
Ans
Global attributes are attributes that can be used with any HTML element. They are not specific to any particular element.Here are 5 global attributes:
accesskey - Specifies a keyboard shortcut for the element.
class - Specifies one or more classes for the element.
contenteditable - Specifies whether the element is editable by the user.
dir - Specifies the direction of the element's content.
draggable - Specifies whether the element is draggable by the user.
